What this tool can and cannot edit
  • Works: PDFs whose text is real, selectable text drawn with embedded or standard fonts using ASCII/Latin-1 encoding (most CAD exports, title blocks, notes, revision tags).
  • Preserved exactly: font, size, position, color, rotation and all other graphics. Only the string operand of the text is changed, written back as an incremental update so the rest of the file is untouched.
  • Batch: the same find/replace is applied to every loaded file independently. One file downloads as a PDF; multiple files download together as a ZIP (you can also download each from the list).
  • Cannot edit: text that the CAD program exported as vector outlines/curves (it is geometry, not text). Use "Show editable text" to confirm what is real text.
  • Limited: subset or CID fonts with custom non-ASCII encodings; replacement characters may lack glyphs in the embedded subset. Prefer characters already used in the drawing.

What this tool can and cannot edit
  • Works: PDFs whose text is real, selectable text drawn with embedded or standard fonts using ASCII/Latin-1 encoding (most CAD exports, title blocks, notes, revision tags).
  • Preserved exactly: font, size, position, color, rotation and all other graphics. Only the string operand of the text is changed, written back as an incremental update so the rest of the file is untouched.
  • Cannot edit: text that the CAD program exported as vector outlines/curves (it is geometry, not text — nothing to find). Use "Show all editable text" to confirm what is real text.
  • Limited: subset or CID fonts that use custom non-ASCII encodings; the new characters may not have glyphs in the embedded subset. The tool reports when text cannot be matched.
  • Glyph note: if you replace with characters the embedded font does not contain (e.g. a symbol), they may render as blanks. Stick to characters already used in the drawing for guaranteed results.
